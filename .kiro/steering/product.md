# Product Overview

Gengar Bark is an AI-powered Slack bot that enhances team communication and productivity at MoeGo Inc.

## Core Capabilities

- **AI Chat**: Natural language conversations powered by OpenAI/Gemini models
- **Context Summarization**: Summarize long Slack threads and discussions
- **CI/CD Integration**: Jenkins build notifications and monitoring via Slack
- **Jira Integration**: Create and manage Jira issues from Slack
- **Appointment Management**: MoeGo appointment scheduling integration
- **HR Bot**: Separate HR-focused bot for employee onboarding and policy queries
- **Event Monitoring**: Track emoji changes, channel events, user status changes

## Bot Instances

- **Gengar Bot**: Primary bot for AI chat, CI/CD, and general productivity
- **HR Bot**: Employee-focused bot for HR policies and onboarding
- **Bob Bot**: Additional bot instance for specific workflows

## Web Interface

Dashboard at pearl.baobo.me for configuration and analytics, including annual reports and user guides.
